# Dialogue Script

**Faris:** We need to fix billing complaints. What should we do?
**Dimas:** Let's define the problem statement first. Can we agree on the scope?
**Faris:** Scope is the billing flow.
**Dimas:** Based on the data, drop-offs increased after the redesign. One possible cause is confusion in checkout.
**Faris:** What are our options?
**Dimas:** The trade-off is speed versus reliability. Given these constraints, I'd recommend a two-week pilot.
**Faris:** What's the main risk?
**Dimas:** The main risk is slower progress if we over-scope. We can mitigate it by setting clear metrics and a strict timeline.

## Audio Direction

- Level: B2
- Speed: slow and natural
- Tone: professional, analytical, calm
- Voices: Dimas and Faris
