# Dialogue Script

**Faris:** Did you read that article about remote work?
**Dimas:** Yes. The article is about how remote work affects productivity.
**Faris:** What's the main point?
**Dimas:** The main point is productivity improves when teams set clear communication rules.
**Faris:** Interesting. Does it mention any downsides?
**Dimas:** Yes, it mentions isolation, especially for new hires.
**Faris:** How does it conclude?
**Dimas:** It concludes that hybrid setups work best for many teams.

## Audio Direction

- Level: B2
- Speed: slow and natural
- Tone: professional, thoughtful, clear
- Voices: Dimas and Faris
