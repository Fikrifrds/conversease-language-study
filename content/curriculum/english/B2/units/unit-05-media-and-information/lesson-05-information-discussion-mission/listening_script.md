# Dialogue Script

**Faris:** Can you summarize the article you read?
**Dimas:** Sure. The article is about remote work and productivity. The main point is clear rules improve focus.
**Faris:** Is the source reliable?
**Dimas:** I'm not sure it's reliable because it doesn't cite data. I'd check official reports.
**Faris:** What's your viewpoint overall?
**Dimas:** From my perspective, hybrid work is a good balance. The reason is it supports focus and connection.
**Faris:** I found a report that contradicts it. The sample size was small.
**Dimas:** I wasn't aware of that. That changes things. I'd like to understand the methodology. Can you share the link so we can revise our conclusion?
**Faris:** Sure. I'll send the link now.

## Audio Direction

- Level: B2
- Speed: slow and natural
- Tone: professional, thoughtful, clear
- Voices: Dimas and Faris
