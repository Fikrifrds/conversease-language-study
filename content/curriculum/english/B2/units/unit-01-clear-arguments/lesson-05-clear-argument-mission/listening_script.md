# Dialogue Script

**Faris:** Should we change our weekly meeting format?
**Dimas:** In my view, we should move routine updates to a written summary.
**Faris:** Why?
**Dimas:** One reason is it saves time. Another reason is it improves focus.
**Faris:** Can you give an example?
**Dimas:** For example, last month we spent 30 minutes repeating status updates.
**Faris:** But we might lose real-time discussion.
**Dimas:** That's a fair point. However, we can keep one monthly live session for deeper discussion.

## Audio Direction

- Level: B2
- Speed: slow and natural
- Tone: professional, clear, confident
- Voices: Dimas and Faris
