# Dialogue Script

**Faris:** Do we really need to invest time in automated tests?
**Dimas:** I think we should. One reason is it reduces bugs in production.
**Faris:** Okay. What's another reason?
**Dimas:** Another reason is it speeds up releases because we can deploy with more confidence.
**Faris:** But writing tests takes time.
**Dimas:** That's true, but it saves time later when we avoid firefighting.
**Faris:** Fair point.
**Dimas:** We can start small and focus on critical flows.

## Audio Direction

- Level: B2
- Speed: slow and natural
- Tone: professional, clear, confident
- Voices: Dimas and Faris
