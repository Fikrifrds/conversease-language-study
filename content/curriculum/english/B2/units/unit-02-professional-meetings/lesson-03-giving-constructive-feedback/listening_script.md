# Dialogue Script

**Ilham:** How did the demo go? Any feedback?
**Faris:** Overall it was clear. I noticed the introduction was a bit long.
**Ilham:** Oh, okay. What was the impact?
**Faris:** Some people lost focus, so the key message came late.
**Ilham:** What would you suggest?
**Faris:** One suggestion is to start with the main takeaway, then show details.
**Ilham:** That makes sense.
**Faris:** It might help if you keep the intro under one minute.

## Audio Direction

- Level: B2
- Speed: slow and natural
- Tone: professional, calm, clear
- Voices: Faris and Ilham
