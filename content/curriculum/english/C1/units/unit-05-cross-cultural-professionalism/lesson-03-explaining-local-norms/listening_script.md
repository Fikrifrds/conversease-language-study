# Dialogue Script

**Ilham:** I'm not sure how direct I should be with the team.
**Faris:** In our context, people tend to be indirect when disagreeing in group settings.
**Ilham:** So I shouldn't disagree openly?
**Faris:** It's generally expected that you raise concerns one-on-one first.
**Ilham:** Interesting. Any tips?
**Faris:** It might help to start with appreciation, then ask a question instead of stating a critique.
**Ilham:** Got it.
**Faris:** That usually lands better and keeps relationships smooth.

## Audio Direction

- Level: C1
- Speed: slow and natural
- Tone: respectful, warm, professional
- Voices: Faris and Ilham
