# Dialogue Script

**Ilham:** How did the last review go?
**Faris:** I wanted to flag one point about the messaging.
**Ilham:** Sure—what is it?
**Faris:** The impact is that some teams interpreted it as a hard deadline.
**Ilham:** That's not what I meant.
**Faris:** I appreciate the intent, and I think we can make it clearer with one sentence.
**Ilham:** What would you suggest?
**Faris:** Would you be open to adding an explicit 'earliest possible' line?
**Ilham:** Yes, that makes sense. Let's add it.

## Audio Direction

- Level: C1
- Speed: slow and natural
- Tone: strategic, calm, professional
- Voices: Faris and Ilham
