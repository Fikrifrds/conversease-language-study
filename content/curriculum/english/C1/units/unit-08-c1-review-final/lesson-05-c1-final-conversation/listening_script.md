# Dialogue Script

**Ilham:** We need a final decision for next quarter.
**Faris:** Let me frame this: we need reliable outcomes and stakeholder confidence before scaling.
**Ilham:** Leadership wants speed.
**Faris:** On balance, we can move fast, but only if we limit scope and validate metrics.
**Ilham:** The team seems hesitant.
**Faris:** What I'm hearing is some fear of visibility. When you say 'hesitant', do you mean scope concerns or accountability concerns?
**Ilham:** Accountability concerns.
**Faris:** Got it. The decision is to run a two-week pilot with clear success criteria. Next steps are: I'll send a one-page plan today, and we'll align stakeholders tomorrow.

## Audio Direction

- Level: C1
- Speed: natural, confident, slightly fast
- Tone: professional, thoughtful, collaborative
- Voices: Faris and Ilham
