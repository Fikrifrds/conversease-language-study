# Challenging an Argument

## Conversation Outcome

After this lesson, learners can challenge an argument constructively using counterexamples, alternative explanations, and precise questions.

## Situation

Kamu menantang argumen orang lain secara konstruktif. Kamu fokus ke logika dan evidence, bukan menyerang personal.

## Lesson Flow

1. Listen to a short dialogue.
2. Understand the conversation with Indonesian support.
3. Practice useful phrases.
4. Repeat key phrases clearly.
5. Respond to short prompts.
6. Practice the same situation with Conversation Coach.
7. Review conversation feedback.
