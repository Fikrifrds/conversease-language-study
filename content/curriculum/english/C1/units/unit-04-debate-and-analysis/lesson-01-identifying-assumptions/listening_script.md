# Dialogue Script

**Faris:** We should cut the onboarding steps to increase conversion.
**Dimas:** It seems you're assuming fewer steps automatically mean better conversion.
**Faris:** Isn't that obvious?
**Dimas:** Not necessarily. What are we assuming about user trust and clarity?
**Faris:** That users just want speed.
**Dimas:** If that's true, then we should see drop-offs mainly on longer forms.
**Faris:** So how do we test it?
**Dimas:** Let's separate essential steps from redundant ones and measure impact.

## Audio Direction

- Level: C1
- Speed: slow and natural
- Tone: analytical, calm, assertive
- Voices: Dimas and Faris
