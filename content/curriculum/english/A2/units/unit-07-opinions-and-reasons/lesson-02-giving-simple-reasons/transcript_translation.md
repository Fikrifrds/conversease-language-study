# Transcript Translation

- **Leo:** Do you like this cafe? -> Kamu suka kafe ini?
- **Mina:** Yes, I like it. -> Iya, aku suka.
- **Leo:** Why? -> Kenapa?
- **Mina:** Because the coffee is good. -> Karena kopinya enak.
- **Leo:** I agree. And it is quiet. -> Setuju. Dan tempatnya tenang.
- **Mina:** Yeah. I come here often because it is relaxing. -> Iya. Aku sering ke sini karena tempatnya santai.
- **Leo:** Nice choice. -> Pilihan bagus.
