# A2 - Everyday Conversations

## Level Outcome

Learners can handle short everyday conversations, ask simple follow-up questions, talk about plans and recent past activities, and respond naturally with basic reactions.

## Skill Weights

| Skill | Weight |
|---|---:|
| Speaking / Conversation Response | 30% |
| Listening Comprehension | 25% |
| Pronunciation & Fluency | 15% |
| Vocabulary / Useful Phrases | 10% |
| Grammar Accuracy | 10% |
| Reading Comprehension | 5% |
| Writing | 5% |

## Passing Threshold

- Overall score: 72
- Speaking / Conversation: 65
- Listening: 65
- Pronunciation: 60
- Vocabulary / Useful Phrases: 65
- Grammar: 65
- Reading: 58
- Writing: 58
- Lesson completion: 80%

Speaking and listening are critical skills. The learner cannot pass A2 if either is below its minimum score.
