# Grammar for Conversation: combining A1 patterns in one conversation

Grammar di lesson ini membantu learner menggabungkan pola A1 yang sudah dipelajari.

## Use short complete sentences.

- Example: My name is Mina. I'm from Indonesia.

## Use question back to keep talking.

- Example: How about you?

## Use polite request forms.

- Example: I would like one tea, please. Can you repeat that, please?
