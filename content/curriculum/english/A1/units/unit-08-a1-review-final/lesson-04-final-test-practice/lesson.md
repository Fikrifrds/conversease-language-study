# Final Test Practice

## Conversation Goal

Learners can answer common A1 test questions about identity, routine, schedule, places, requests, and simple needs.

## Situation Setup

Ini latihan sebelum final test. Kamu menjawab beberapa pertanyaan pendek dari examiner dengan kalimat sederhana dan jelas.

## Listen First

Dengarkan dialog review ini. Fokus pada cara menggabungkan skill A1 lama dalam percakapan yang tetap pendek dan natural.

## Key Conversation Flow

1. Introduce yourself.
2. Answer one origin question.
3. Talk about one routine and time.
4. Ask for repetition if needed.
5. Make one polite request.

## Practice Target

By the end, you should be able to say: "My name is Alya. I'm from Indonesia. I study English at seven."
