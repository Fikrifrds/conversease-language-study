# Grammar for Conversation: imperatives for directions

Grammar di lesson ini hanya dipakai untuk membantu percakapan, bukan untuk hafalan panjang.

## Direction commands can start with the verb.

- Example: Go straight. Turn left.

## Use 'then' for the next step.

- Example: Then turn left.

## Use 'on the left/right' for side location.

- Example: The room is on the right.
