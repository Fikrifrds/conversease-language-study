# Grammar for Conversation: going to + place

Grammar di lesson ini hanya dipakai untuk membantu percakapan, bukan untuk hafalan panjang.

## Use 'I am going to + place.'

- Example: I'm going to the cafe.

## Use 'the' for a known place.

- Example: the cafe, the library

## Ask 'Is it near here?'

- Example: Is the cafe near here?
