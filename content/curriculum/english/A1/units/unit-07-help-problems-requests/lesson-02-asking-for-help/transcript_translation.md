# Transcript Translation

- **Dimas:** Excuse me. Can you help me? -> Permisi. Bisa bantu saya?
- **Omar:** Sure. What is the problem? -> Tentu. Apa masalahnya?
- **Dimas:** I can't open this file. -> Saya tidak bisa membuka file ini.
- **Omar:** Okay. Click this button. -> Oke. Klik tombol ini.
- **Dimas:** This button? -> Tombol ini?
- **Omar:** Yes. Try again. -> Ya. Coba lagi.
- **Dimas:** It works. Thank you. -> Berhasil. Terima kasih.
