# Grammar for Conversation: can / can't

Grammar di lesson ini membantu learner meminta bantuan dan menjelaskan masalah secara singkat.

## Use 'Can you help me?' to ask for help.

- Example: Can you help me?

## Use 'I can't + verb' for a simple problem.

- Example: I can't open this file.

## Use 'It works' when the problem is fixed.

- Example: It works.
