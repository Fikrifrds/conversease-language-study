# Asking for Help

## Conversation Goal

Learners can ask for help with one simple task and respond to the helper.

## Situation Setup

Kamu kesulitan membuka file kelas. Minta bantuan singkat dan sopan.

## Listen First

Dengarkan dialog pendek. Fokus pada cara menjelaskan masalah, meminta bantuan, dan merespons dengan sopan.

## Key Conversation Flow

1. Say the problem or confusion clearly.
2. Ask for help or repetition.
3. Confirm the instruction or request.
4. Respond when the problem is fixed.
5. Close with thanks or apology.

## Practice Target

By the end, you should be able to say: "Can you help me?"
