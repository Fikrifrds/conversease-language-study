# Grammar for Conversation: combining help language

Grammar di lesson ini membantu learner meminta bantuan dan menjelaskan masalah secara singkat.

## Use 'I don't understand' for meaning problems.

- Example: Sorry, I don't understand.

## Use 'I can't + verb' for action problems.

- Example: I can't open this file.

## Use 'Can you + verb?' for requests.

- Example: Can you wait a minute?
