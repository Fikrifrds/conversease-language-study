# A1 - Start Simple Conversations

## Level Outcome

Learners can start and close simple conversations, introduce themselves, ask and answer simple personal questions, understand short slow dialogues, and respond without freezing.

## Skill Weights

| Skill | Weight |
|---|---:|
| Speaking / Conversation Response | 30% |
| Listening Comprehension | 25% |
| Pronunciation & Fluency | 15% |
| Vocabulary / Useful Phrases | 10% |
| Grammar Accuracy | 10% |
| Reading Comprehension | 5% |
| Writing | 5% |

## Passing Threshold

- Overall score: 70
- Speaking / Conversation: 60
- Listening: 60
- Pronunciation: 55
- Vocabulary / Useful Phrases: 60
- Grammar: 60
- Reading: 55
- Writing: 55
- Lesson completion: 80%

Speaking and listening are critical skills. The learner cannot pass A1 if either is below its minimum score.
