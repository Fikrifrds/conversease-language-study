# B1 - Confident Everyday Speaking

## Level Outcome

Learners can tell connected personal stories, handle common workplace and travel situations, explain problems and solutions, and share opinions with clear reasons in everyday conversations.

## Skill Weights

| Skill | Weight |
|---|---:|
| Speaking / Conversation Response | 35% |
| Listening Comprehension | 25% |
| Pronunciation & Fluency | 15% |
| Vocabulary / Useful Phrases | 10% |
| Grammar Accuracy | 10% |
| Reading Comprehension | 3% |
| Writing | 2% |

## Passing Threshold

- Overall score: 75
- Speaking / Conversation: 70
- Listening: 68
- Pronunciation: 62
- Vocabulary / Useful Phrases: 68
- Grammar: 68
- Reading: 60
- Writing: 60
- Lesson completion: 80%

Speaking and listening are critical skills. The learner cannot pass B1 if either is below its minimum score.
