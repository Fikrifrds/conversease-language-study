# Dialogue Script

**Faris:** So which restaurant should we choose?  
**Mina:** How about we go with the cheaper one tonight?  
**Faris:** I'm okay with that, but is it too crowded?  
**Mina:** It might be, so let's go early.  
**Faris:** That works for me.  
**Mina:** Great. So we agree on Noodle House at 6:30?  
**Faris:** Yes, let's do that.  
**Mina:** Perfect.  

## Audio Direction

- Level: B1
- Speed: slow and natural
- Tone: friendly, clear, helpful
- Voices: Mina and Faris
