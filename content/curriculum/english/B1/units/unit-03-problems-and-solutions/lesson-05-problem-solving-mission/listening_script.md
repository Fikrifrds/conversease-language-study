# Dialogue Script

**Faris:** Quick check: what's the issue?
**Dimas:** There's a problem with the login page. Users can't sign in.
**Faris:** When did it start?
**Dimas:** It started this morning after the deploy, so it's urgent.
**Faris:** What should we do?
**Dimas:** We could roll back now because it's quick.
**Faris:** But will it affect other features?
**Dimas:** It might, so let's roll back and then test the key flows.

## Audio Direction

- Level: B1
- Speed: slow and natural
- Tone: friendly, clear, helpful
- Voices: Faris and Dimas
