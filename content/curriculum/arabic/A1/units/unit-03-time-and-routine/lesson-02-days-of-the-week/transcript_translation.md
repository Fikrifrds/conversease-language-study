# Transcript Translation

- **Noura:** أي يَوْمَ الْيَوْمُ؟ -> Hari apa hari ini?
- **Ahmad:** الْيَوْمُ الْإِثْنَيْنِ. -> Hari ini Senin.
- **Noura:** هَلْ عِنْدَكَ دَرْسٌ غَدًا؟ -> Apakah kamu punya pelajaran besok?
- **Ahmad:** نَعَمْ، عِنْدِي دَرْسٌ يَوْمَ الثُّلَاثَاءِ. -> Ya, saya punya pelajaran hari Selasa.
- **Noura:** جَيِّدٌ. -> Bagus.
