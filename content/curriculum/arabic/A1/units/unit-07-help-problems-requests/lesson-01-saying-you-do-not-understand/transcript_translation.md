# Transcript Translation

- **Maryam:** اقرأ الجملة. -> Bacalah kalimatnya.
- **Ahmad:** لا أفهم الكلمة. -> Saya tidak paham katanya.
- **Maryam:** أي كلمة؟ -> Kata yang mana?
- **Ahmad:** ماذا يعني هذا؟ -> Apa arti ini?
- **Maryam:** يعني: كتاب. -> Artinya: buku.
- **Ahmad:** شكرًا، الآن أفهم. -> Terima kasih, sekarang saya paham.
