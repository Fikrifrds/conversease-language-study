# Transcript Translation

- **Muallim:** افتح الكتاب من فضلك. -> Buka bukunya, tolong.
- **Layla:** نعم، أفتح الكتاب. -> Ya, saya buka bukunya.
- **Muallim:** اكتبي الجملة. -> Tulis kalimatnya.
- **Layla:** أكتب الجملة الآن. -> Saya menulis kalimatnya sekarang.
- **Muallim:** استمعي ثم تكلمي. -> Dengarkan lalu bicaralah.
- **Layla:** حسنًا. -> Baik.
