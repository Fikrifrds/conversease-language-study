# Transcript Translation

- **Muallim:** اِفْتَحِ الْكِتَابَ مِنْ فَضْلِكَ. -> Buka bukunya, tolong.
- **Rami:** نَعَمْ، أَفْتَحُ الْكِتَابَ. -> Ya, saya buka bukunya.
- **Muallim:** اُكْتُبِ الْجُمْلَةَ. -> Tulis kalimatnya.
- **Rami:** أَكْتُبُ الْجُمْلَةَ الْآنَ. -> Saya menulis kalimatnya sekarang.
- **Muallim:** اِسْتَمِعْ ثُمَّ تَكَلَّمْ. -> Dengarkan lalu bicaralah.
- **Rami:** حَسَنًا. -> Baik.
