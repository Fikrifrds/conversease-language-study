# Transcript Translation

- **Maryam:** أَيْنَ تَدْرُسُ؟ -> Di mana kamu belajar?
- **Layla:** أَدْرُسُ فِي الْمَرْكَزِ. -> Saya belajar di pusat belajar.
- **Maryam:** مَاذَا تَدْرُسِينَ؟ -> Apa yang kamu pelajari?
- **Layla:** أَدْرُسُ اللُّغَةَ الْعَرَبِيَّةَ. -> Saya belajar bahasa Arab.
- **Maryam:** هل تَعْمَلِينَ أيضًا؟ -> Apakah kamu juga bekerja?
- **Layla:** نَعَمْ، أَعْمَلُ. -> Ya, saya bekerja.
