# Transcript Translation

- **Muallim:** إِلَى أَيْنَ تُرِيدُ أَنْ تَذْهَبَ؟ -> Ke mana kamu ingin pergi?
- **Zayd:** أُرِيدُ أَنْ أَذْهَبَ إِلَى الْفَصْلِ. -> Saya ingin pergi ke kelas.
- **Muallim:** اِذْهَبِي يَمِينًا ثُمَّ إِلَى الْأَمَامِ. -> Pergilah ke kanan lalu lurus ke depan.
- **Zayd:** وأَيْنَ الْفَصْلُ؟ -> Dan di mana kelasnya?
- **Muallim:** الْفَصْلُ بِجَانِبِ الْمَكْتَبَةِ. -> Kelasnya di samping perpustakaan.
- **Zayd:** حَسَنًا، شُكْرًا. -> Baik, terima kasih.
