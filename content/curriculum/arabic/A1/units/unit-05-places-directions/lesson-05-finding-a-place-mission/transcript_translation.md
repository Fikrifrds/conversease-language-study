# Transcript Translation

- **Muallim:** إِلَى أَيْنَ تُرِيدِينَ أَنْ تَذْهَبِي؟ -> Ke mana kamu ingin pergi?
- **Noura:** أُرِيدُ أَنْ أَذْهَبَ إِلَى الْفَصْلِ. -> Saya ingin pergi ke kelas.
- **Muallim:** اِذْهَبِي يَمِينًا ثُمَّ إِلَى الْأَمَامِ. -> Pergilah ke kanan lalu lurus ke depan.
- **Noura:** وأَيْنَ الْفَصْلُ؟ -> Dan di mana kelasnya?
- **Muallim:** الْفَصْلُ بِجَانِبِ الْمَكْتَبَةِ. -> Kelasnya di samping perpustakaan.
- **Noura:** حَسَنًا، شُكْرًا. -> Baik, terima kasih.
