# Transcript Translation

- **Zayd:** أَيْنَ الْمَدْرَسَةُ؟ -> Di mana sekolah?
- **Ahmad:** الْمَدْرَسَةُ قَرِيبَةٌ. -> Sekolah dekat.
- **Zayd:** وَأَيْنَ السُّوقُ؟ -> Dan di mana pasar?
- **Ahmad:** السُّوقُ بَعِيدٌ. -> Pasar jauh.
- **Zayd:** هَلِ الْمَقْهَى هُنَا؟ -> Apakah kafe di sini?
- **Ahmad:** نَعَمْ، الْمَقْهَى هُنَا. -> Ya, kafe di sini.
