# Transcript Translation

- **Khalid:** كَيْفَ أَذْهَبُ إِلَى الْمَكْتَبَةِ؟ -> Bagaimana saya pergi ke perpustakaan?
- **Rami:** اِذْهَبْ إِلَى الْأَمَامِ. -> Pergilah ke depan.
- **Khalid:** ثُمَّ؟ -> Lalu?
- **Rami:** ثُمَّ اِذْهَبْ يَمِينًا. -> Lalu pergilah ke kanan.
- **Khalid:** أَيْنَ الْمَكْتَبَةُ؟ -> Di mana perpustakaan?
- **Rami:** هِيَ بِجَانِبِ الْمَكْتَبِ. -> Perpustakaan di samping kantor.
