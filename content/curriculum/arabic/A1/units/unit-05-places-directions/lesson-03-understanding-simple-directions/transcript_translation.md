# Transcript Translation

- **Khalid:** كيف أذهب إلى المكتبة؟ -> Bagaimana saya pergi ke perpustakaan?
- **Layla:** اذهب إلى الأمام. -> Pergilah ke depan.
- **Khalid:** ثم؟ -> Lalu?
- **Layla:** ثم اذهب يمينًا. -> Lalu pergilah ke kanan.
- **Khalid:** أين المكتبة؟ -> Di mana perpustakaan?
- **Layla:** هي بجانب المكتب. -> Perpustakaan di samping kantor.
