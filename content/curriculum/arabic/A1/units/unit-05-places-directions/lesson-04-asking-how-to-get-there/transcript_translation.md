# Transcript Translation

- **Ahmad:** كَيْفَ أَذْهَبُ إِلَى الْمَقْهَى؟ -> Bagaimana saya pergi ke kafe?
- **Maryam:** امشِ إِلَى الْأَمَامِ. -> Berjalanlah ke depan.
- **Ahmad:** ثُمَّ مَاذَا؟ -> Lalu apa?
- **Maryam:** ثُمَّ اِذْهَبْ يَسَارًا عِنْدَ الْبَابِ. -> Lalu pergilah ke kiri di dekat pintu.
- **Ahmad:** شُكْرًا. -> Terima kasih.
