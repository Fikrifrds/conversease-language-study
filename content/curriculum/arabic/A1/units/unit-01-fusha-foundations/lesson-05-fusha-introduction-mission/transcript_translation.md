# Transcript Translation

- **Muallim:** مَرْحَبًا، مَا اسْمُكَ؟ -> Halo, siapa namamu?
- **Ahmad:** مَرْحَبًا، اِسْمِي أَحْمَدُ. -> Halo, nama saya Ahmad.
- **Muallim:** مِنْ أَيْنَ أَنْتَ؟ -> Dari mana kamu?
- **Ahmad:** أَنَا مِنْ إِنْدُونِيسِيَا. -> Saya dari Indonesia.
- **Muallim:** اِقْرَأِ الْجُمْلَةَ. -> Bacalah kalimat itu.
- **Ahmad:** عُفْوًا، لَا أَفْهَمُ. أَعِدْ مِنْ فَضْلِكَ. -> Maaf, saya tidak paham. Ulangi, tolong.
- **Muallim:** حَسَنًا، أَقْرَؤُهَا مَرَّةً أُخْرَى. -> Baik, saya membacanya sekali lagi.
