# Transcript Translation

- **Muallim:** مَا اسْمُكَ؟ -> Siapa namamu?
- **Ahmad:** اِسْمِي أَحْمَدُ. -> Nama saya Ahmad.
- **Muallim:** مِنْ أَيْنَ أَنْتَ؟ -> Dari mana kamu?
- **Ahmad:** أَنَا مِنْ إِنْدُونِيسِيَا. وَأَنْتَ؟ -> Saya dari Indonesia. Dan Anda?
- **Muallim:** أَنَا مِنَ الْمَمْلَكَةِ الْعَرَبِيَّةِ السُّعُودِيَّةِ. -> Saya dari Kerajaan Arab Saudi.
