# Transcript Translation

- **Muallim:** اِقْرَأِ الْجُمْلَةَ. -> Bacalah kalimat itu.
- **Ahmad:** نَعَمْ يَا مُعَلِّمُ. -> Baik, Pak Guru.
- **Muallim:** اِسْتَمِعْ ثُمَّ أَعِدْ. -> Dengarkan, lalu ulangi.
- **Ahmad:** نَعَمْ، أُعِيدُ الْجُمْلَةَ. -> Ya, saya ulangi kalimatnya.
- **Muallim:** أَحْسَنْتَ. اُكْتُبِ الْكَلِمَةَ الْآنَ. -> Bagus. Tulislah kata itu sekarang.
