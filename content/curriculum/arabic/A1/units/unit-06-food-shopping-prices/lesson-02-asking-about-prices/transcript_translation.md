# Transcript Translation

- **Maryam:** كَمْ سِعْرُ الْقَلَمِ؟ -> Berapa harga penanya?
- **Zayd:** سِعْرُهُ خَمْسَةُ رِيَالَاتٍ. -> Harganya lima riyal.
- **Maryam:** السِّعْرُ مُنَاسِبٌ. -> Harganya cocok.
- **Zayd:** هَلْ تَأْخُذِينَ وَاحِدًا؟ -> Apakah kamu mengambil satu?
- **Maryam:** نَعَمْ، آخُذُ وَاحِدًا. -> Ya, saya ambil satu.
