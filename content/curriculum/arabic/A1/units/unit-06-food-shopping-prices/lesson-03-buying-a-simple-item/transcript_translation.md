# Transcript Translation

- **Ahmad:** هَلْ عِنْدَكُمْ قَلَمٌ؟ -> Apakah kalian punya pena?
- **Noura:** نَعَمْ، مَوْجُودٌ. -> Ya, ada.
- **Ahmad:** أُرِيدُ هَذَا. -> Saya ingin ini.
- **Noura:** كَمْ قَلَمًا تُرِيدُ؟ -> Berapa pena yang kamu inginkan?
- **Ahmad:** آخُذُ وَاحِدًا. -> Saya ambil satu.
