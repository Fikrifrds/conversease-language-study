# Transcript Translation

- **Muallimah:** ما اسمك؟ -> Siapa namamu?
- **Ahmad:** اسمي أحمد. -> Nama saya Ahmad.
- **Muallimah:** كيف تكتب اسمك؟ -> Bagaimana kamu menulis namamu?
- **Ahmad:** أكتب اسمي: أحمد. -> Saya menulis nama saya: Ahmad.
- **Muallimah:** ما رقم هاتفك؟ -> Berapa nomor teleponmu?
- **Ahmad:** رقمي واحد، اثنان، ثلاثة، أربعة. -> Nomor saya satu, dua, tiga, empat.
- **Muallimah:** هل هذا واضح؟ -> Apakah ini jelas?
- **Ahmad:** نعم، واضح. -> Ya, jelas.
