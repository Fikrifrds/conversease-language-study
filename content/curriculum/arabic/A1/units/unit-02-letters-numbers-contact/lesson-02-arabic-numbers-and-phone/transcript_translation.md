# Transcript Translation

- **Maryam:** مَا رَقْمُ هَاتِفِكَ؟ -> Berapa nomor teleponmu?
- **Khalid:** رَقْمِي وَاحِدٌ، اِثْنَانِ، ثَلَاثَةٌ، أَرْبَعَةٌ. -> Nomor saya satu, dua, tiga, empat.
- **Maryam:** هَلِ الرَّقْمُ وَاحِدٌ، اِثْنَانِ، ثَلَاثَةٌ، أَرْبَعَةٌ؟ -> Apakah nomornya satu, dua, tiga, empat?
- **Khalid:** نَعَمْ، الرَّقْمُ صَحِيحٌ. -> Ya, nomornya benar.
- **Maryam:** شُكْرًا، حَفِظْتُ الرَّقْمَ. -> Terima kasih, saya simpan nomornya.
