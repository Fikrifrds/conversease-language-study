# Transcript Translation

- **Layla:** لم أسمع الحرف. -> Saya tidak mendengar hurufnya.
- **Zayd:** الْحَرْفُ بَاءٌ. -> Hurufnya ba.
- **Layla:** بِبُطْءٍ مِنْ فَضْلِكَ. -> Pelan-pelan, tolong.
- **Zayd:** بَاءٌ. -> Ba.
- **Layla:** هَلْ قُلْتَ بَاءً؟ -> Apakah Anda mengatakan ba?
- **Zayd:** نَعَمْ، قلت بَاءٌ. -> Ya, saya mengatakan ba.
