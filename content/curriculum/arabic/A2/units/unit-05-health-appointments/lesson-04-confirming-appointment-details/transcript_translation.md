# Transcript Translation

- **Rami:** مَرْحَبًا، أُرِيدُ أَنْ أُؤَكِّدَ الْمَوْعِدَ. -> Halo, saya ingin mengonfirmasi janji.
- **Khalid:** بِاسْمِ مَنْ؟ -> Atas nama siapa?
- **Rami:** بِاسْمِ رَامِي. -> Atas nama Rami.
- **Khalid:** نَعَمْ، مَوْعِدُكَ غَدًا السَّاعَةَ التَّاسِعَةَ. -> Ya, janjimu besok jam sembilan.
- **Rami:** فِي أَيِّ طَابِقٍ؟ -> Di lantai berapa?
- **Khalid:** فِي الطَّابِقِ الثَّانِي، غُرْفَةُ الِاسْتِقْبَالِ. -> Di lantai dua, ruang penerimaan.
- **Rami:** شُكْرًا. التَّفَاصِيلُ صَحِيحَةٌ. -> Terima kasih. Detailnya benar.
