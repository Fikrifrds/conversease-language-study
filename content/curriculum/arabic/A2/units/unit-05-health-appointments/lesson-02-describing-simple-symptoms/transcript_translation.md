# Transcript Translation

- **Omar:** عِنْدِي أَلَمٌ فِي الْحَلْقِ. -> Saya punya sakit di tenggorokan.
- **Hakim:** مُنْذُ مَتَى؟ -> Sejak kapan?
- **Omar:** مُنْذُ يَوْمَيْنِ. -> Sejak dua hari.
- **Hakim:** هَلْ عِنْدَكَ سُعَالٌ؟ -> Apakah kamu punya batuk?
- **Omar:** نَعَمْ، لَدَيَّ سُعَالٌ خَفِيفٌ. -> Ya, saya punya batuk ringan.
- **Hakim:** وَدَرَجَةُ الْحَرَارَةِ؟ -> Dan suhu badan?
- **Omar:** دَرَجَةُ حَرَارَتِي مُرْتَفِعَةٌ قَلِيلًا، وَلَكِنَّ الْأَلَمَ لَيْسَ شَدِيدًا. -> Suhu badan saya sedikit tinggi, tetapi sakitnya tidak berat.
