# Target Percakapan

Latih percakapan Arab A2 untuk situasi ini: Kamu menelepon klinik untuk membuat janji, memilih hari dan waktu, serta menyebut nama untuk pendaftaran.

Kamu akan berlatih mengatakan:

- أُرِيدُ مَوْعِدًا مَعَ الطَّبِيبِ.
- هَلْ يُوجَدُ مَوْعِدٌ غَدًا؟
- صَبَاحًا أَمْ مَسَاءً؟
- السَّاعَةُ الْعَاشِرَةُ مُنَاسِبَةٌ.
- الْمَوْعِدُ بِاسْمِي.
