# Transcript Translation

- **Noura:** مَرْحَبًا، أُرِيدُ مَوْعِدًا مَعَ الطَّبِيبِ. -> Halo, saya ingin janji dengan dokter.
- **Fatimah:** مَرْحَبًا. هَلْ تُرِيدِينَ الْمَوْعِدَ غَدًا؟ -> Halo. Apakah kamu ingin janji besok?
- **Noura:** نَعَمْ، هَلْ يُوجَدُ مَوْعِدٌ فِي الصَّبَاحِ؟ -> Ya, apakah ada janji pada pagi hari?
- **Fatimah:** يُوجَدُ مَوْعِدٌ السَّاعَةَ الْعَاشِرَةَ. -> Ada janji jam sepuluh.
- **Noura:** السَّاعَةُ الْعَاشِرَةُ مُنَاسِبَةٌ. -> Jam sepuluh cocok.
- **Fatimah:** بِاسْمِ مَنْ؟ -> Atas nama siapa?
- **Noura:** الْمَوْعِدُ بِاسْمِ نُورَةَ. -> Janji atas nama Noura.
