# Transcript Translation

- **Sara:** مَرْحَبًا، لَا أَشْعُرُ بِالرَّاحَةِ. -> Halo, saya tidak merasa nyaman.
- **Fatimah:** مَا الَّذِي تَشْعُرِينَ بِهِ؟ -> Apa yang kamu rasakan?
- **Sara:** عِنْدِي أَلَمٌ خَفِيفٌ فِي الْحَلْقِ مُنْذُ أَمْسِ. -> Saya punya sakit ringan di tenggorokan sejak kemarin.
- **Fatimah:** هَلْ تُرِيدِينَ مَوْعِدًا مَعَ الطَّبِيبِ؟ -> Apakah kamu ingin janji dengan dokter?
- **Sara:** نَعَمْ، هَلْ يُوجَدُ مَوْعِدٌ الْيَوْمَ؟ -> Ya, apakah ada janji hari ini?
- **Fatimah:** يُوجَدُ مَوْعِدٌ السَّاعَةَ الْحَادِيَةَ عَشْرَةَ. -> Ada janji jam sebelas.
- **Sara:** السَّاعَةُ الْحَادِيَةَ عَشْرَةَ مُنَاسِبَةٌ. -> Jam sebelas cocok.
- **Fatimah:** الْمَوْعِدُ بِاسْمِ سَارَةَ فِي الطَّابِقِ الثَّانِي. -> Janji atas nama Sara di lantai dua.
- **Sara:** شُكْرًا. أُرِيدُ أَنْ أُؤَكِّدَ التَّفَاصِيلَ: الْيَوْمَ، السَّاعَةَ الْحَادِيَةَ عَشْرَةَ، الطَّابِقَ الثَّانِي. -> Terima kasih. Saya ingin mengonfirmasi detail: hari ini, jam sebelas, lantai dua.
- **Fatimah:** نَعَمْ، التَّفَاصِيلُ صَحِيحَةٌ. -> Ya, detailnya benar.
