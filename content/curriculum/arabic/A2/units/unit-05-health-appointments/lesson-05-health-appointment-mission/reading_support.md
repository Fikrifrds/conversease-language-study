# Bantuan Membaca

Baca frasa Arab dari kanan ke kiri. Kenali pola utuhnya dulu, lalu perhatikan kata tanya, alasan, atau kata kerja lampau/dekat.

- لَا أَشْعُرُ بِالرَّاحَةِ. -> Saya tidak merasa nyaman.
- عِنْدِي أَلَمٌ خَفِيفٌ. -> Saya punya sakit ringan.
- أُرِيدُ مَوْعِدًا مَعَ الطَّبِيبِ. -> Saya ingin janji dengan dokter.
- السَّاعَةُ الْحَادِيَةَ عَشْرَةَ مُنَاسِبَةٌ. -> Jam sebelas cocok.
- أُرِيدُ أَنْ أُؤَكِّدَ التَّفَاصِيلَ. -> Saya ingin mengonfirmasi detail.
