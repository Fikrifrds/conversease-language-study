# Transcript Translation

- **Layla:** أَيُّ مَكَانٍ تُفَضِّلُ لِلدِّرَاسَةِ يَا رَامِي؟ -> Tempat mana yang kamu pilih untuk belajar, Rami?
- **Rami:** أُفَضِّلُ الْمَكْتَبَةَ. -> Saya lebih memilih perpustakaan.
- **Layla:** لِمَاذَا؟ -> Mengapa?
- **Rami:** السَّبَبُ بَسِيطٌ. أُفَضِّلُهَا لِأَنَّهَا هَادِئَةٌ وَقَرِيبَةٌ. -> Alasannya sederhana. Saya memilihnya karena tenang dan dekat.
- **Layla:** وَالْمَقْهَى؟ -> Dan kafe?
- **Rami:** الْمَقْهَى جَيِّدٌ، وَلَكِنَّهُ مُزْدَحِمٌ. -> Kafe bagus, tetapi ramai.
- **Layla:** فَهِمْتُ. الْمَكْتَبَةُ أَفْضَلُ لَكَ. -> Saya mengerti. Perpustakaan lebih baik untukmu.
