# Transcript Translation

- **Khalid:** مَا رَأْيُكِ فِي هَذِهِ الْخُطَّةِ يَا لَيْلَى؟ -> Apa pendapatmu tentang rencana ini, Layla?
- **Layla:** أَعْتَقِدُ أَنَّهَا جَيِّدَةٌ، وَلَكِنَّ الْوَقْتَ غَيْرُ مُنَاسِبٍ. -> Saya pikir itu baik, tetapi waktunya tidak cocok.
- **Khalid:** أَيُّ وَقْتٍ تُفَضِّلِينَ؟ -> Waktu apa yang kamu pilih?
- **Layla:** أُفَضِّلُ الصَّبَاحَ لِأَنَّهُ أَهْدَأُ. -> Saya lebih memilih pagi karena lebih tenang.
- **Khalid:** لَا أُوَافِقُ تَمَامًا. الصَّبَاحُ صَعْبٌ لِي. -> Saya tidak sepenuhnya setuju. Pagi sulit untuk saya.
- **Layla:** أَفْهَمُ رَأْيَكَ. مَاذَا عَنِ السَّاعَةِ الْعَاشِرَةِ؟ -> Saya memahami pendapatmu. Bagaimana dengan jam sepuluh?
- **Khalid:** السَّاعَةُ الْعَاشِرَةُ مُنَاسِبَةٌ. -> Jam sepuluh cocok.
- **Layla:** إِذًا نَخْتَارُ هَذَا. -> Jadi kita memilih ini.
