# Transcript Translation

- **Maryam:** هَلْ تُرِيدِينَ أَنْ تَدْرُسِي مَعِي الْيَوْمَ؟ -> Apakah kamu ingin belajar bersama saya hari ini?
- **Noura:** رُبَّمَا. لِمَاذَا؟ -> Mungkin. Mengapa?
- **Maryam:** لِأَنَّ الدَّرْسَ مُهِمٌّ، وَالِاخْتِبَارَ قَرِيبٌ. -> Karena pelajarannya penting, dan tesnya dekat.
- **Noura:** فِكْرَةٌ جَيِّدَةٌ. أَنَا مَوْجُودَةٌ بَعْدَ الْعَصْرِ. -> Ide yang bagus. Saya ada setelah Asar.
- **Maryam:** جَيِّدٌ. نَدْرُسُ فِي الْمَرْكَزِ. -> Bagus. Kita belajar di pusat.
- **Noura:** حَسَنًا، سَأَحْضُرُ. -> Baik, saya akan hadir.
