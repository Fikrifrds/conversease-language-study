# Transcript Translation

- **Khalid:** هَلْ تُرِيدُ أَنْ تَدْرُسَ مَعِي الْيَوْمَ؟ -> Apakah kamu ingin belajar bersama saya hari ini?
- **Raka:** نَعَمْ، وَلَكِنْ لِمَاذَا الْيَوْمَ؟ -> Ya, tetapi mengapa hari ini?
- **Khalid:** لِأَنَّ الِاخْتِبَارَ قَرِيبٌ. -> Karena tesnya dekat.
- **Raka:** حَسَنًا. نَلْتَقِي السَّاعَةَ الرَّابِعَةَ؟ -> Baik. Kita bertemu jam empat?
- **Khalid:** آسِفَةٌ، هَلْ يُمْكِنُ أَنْ نُغَيِّرَ الْمَوْعِدَ؟ -> Maaf, bisakah kita mengubah jadwalnya?
- **Raka:** نَعَمْ. مَاذَا عَنِ السَّاعَةِ الْخَامِسَةِ؟ -> Ya. Bagaimana dengan jam lima?
- **Khalid:** جَيِّدٌ. نَلْتَقِي أَمَامَ الْمَرْكَزِ. -> Bagus. Kita bertemu di depan pusat.
- **Raka:** حَسَنًا، إِلَى اللِّقَاءِ. -> Baik, sampai bertemu.
