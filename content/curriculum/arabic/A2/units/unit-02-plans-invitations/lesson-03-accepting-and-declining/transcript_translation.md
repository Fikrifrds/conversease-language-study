# Transcript Translation

- **Omar:** هَلْ تَأْتِي إِلَى الْمَكْتَبَةِ مَعِي؟ -> Apakah kamu datang ke perpustakaan bersama saya?
- **Ahmad:** نَعَمْ، أَسْتَطِيعُ بَعْدَ الظُّهْرِ. -> Ya, saya bisa setelah Zuhur.
- **Omar:** وَهَلْ تَأْتِي إِلَى السُّوقِ مَسَاءً؟ -> Dan apakah kamu datang ke pasar sore?
- **Ahmad:** آسِفٌ، لَا أَسْتَطِيعُ. -> Maaf, saya tidak bisa.
- **Omar:** لِمَاذَا؟ -> Mengapa?
- **Ahmad:** لِأَنَّ لَدَيَّ مَوْعِدًا مَعَ أُسْرَتِي. -> Karena saya punya janji dengan keluarga saya.
- **Omar:** لَا مُشْكِلَةَ. رُبَّمَا فِي وَقْتٍ آخَرَ. -> Tidak masalah. Mungkin di waktu lain.
