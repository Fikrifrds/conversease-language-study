# Transcript Translation

- **Khalid:** سَأَذْهَبُ إِلَى الْمَكْتَبَةِ بَعْدَ الدَّرْسِ. -> Saya akan pergi ke perpustakaan setelah pelajaran.
- **Zayd:** هَلْ تُرِيدُ أَنْ تَذْهَبَ مَعِي؟ -> Apakah kamu ingin pergi bersama saya?
- **Khalid:** نَعَمْ، أُرِيدُ ذَلِكَ. مَتَى نَلْتَقِي؟ -> Ya, saya ingin itu. Kapan kita bertemu?
- **Zayd:** نَلْتَقِي السَّاعَةَ الرَّابِعَةَ. -> Kita bertemu jam empat.
- **Khalid:** أَيْنَ نَلْتَقِي؟ -> Di mana kita bertemu?
- **Zayd:** أَمَامَ الْمَكْتَبَةِ. -> Di depan perpustakaan.
- **Khalid:** حَسَنًا، إِلَى اللِّقَاءِ. -> Baik, sampai bertemu.
