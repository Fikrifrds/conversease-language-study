# Transcript Translation

- **Zayd:** عَفْوًا، كَيْفَ أَذْهَبُ إِلَى الرَّصِيفِ الثَّالِثِ؟ -> Permisi, bagaimana saya pergi ke peron tiga?
- **Karim:** اِمْشِي مُسْتَقِيمًا إِلَى نِهَايَةِ الْمَمَرِّ. -> Berjalanlah lurus sampai ujung lorong.
- **Zayd:** ثُمَّ؟ -> Lalu?
- **Karim:** ثُمَّ اِتَّجِهِي يَمِينًا. الرَّصِيفُ بِجَانِبِ الْمَصْعَدِ. -> Lalu belok kanan. Peronnya di samping lift.
- **Zayd:** أَمْشِي مُسْتَقِيمًا، ثُمَّ أَتَّجِهُ يَمِينًا. هَلْ هَذَا صَحِيحٌ؟ -> Saya berjalan lurus, lalu belok kanan. Apakah ini benar?
- **Karim:** نَعَمْ، هَذَا صَحِيحٌ. -> Ya, itu benar.
