# Transcript Translation

- **Rami:** ذَهَبْتُ إِلَى مَكْتَبَةٍ جَدِيدَةٍ أَمْسِ. -> Saya pergi ke perpustakaan baru kemarin.
- **Adi:** حَقًّا؟ كَيْفَ كَانَ الْمَكَانُ؟ -> Benarkah? Bagaimana tempatnya?
- **Rami:** كَانَ نَظِيفًا وَهَادِئًا. -> Tempatnya bersih dan tenang.
- **Adi:** هَذَا مُمْتَازٌ. هَلْ هِيَ قَرِيبَةٌ؟ -> Itu luar biasa. Apakah dekat?
- **Rami:** نَعَمْ، هِيَ قَرِيبَةٌ مِنَ الْمَرْكَزِ. -> Ya, dekat dari pusat.
- **Adi:** أُرِيدُ أَنْ أَزُورَ ذَلِكَ الْمَكَانَ. -> Saya ingin mengunjungi tempat itu.
