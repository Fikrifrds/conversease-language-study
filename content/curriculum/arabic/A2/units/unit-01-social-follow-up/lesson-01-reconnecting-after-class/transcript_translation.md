# Transcript Translation

- **Khalid:** السَّلَامُ عَلَيْكَ يَا كَرِيمُ. -> Assalamu'alaikum, Karim.
- **Karim:** وَعَلَيْكُمُ السَّلَامُ يَا خَالِدُ. كَيْفَ حَالُكَ؟ -> Wa'alaikumussalam, Khalid. Apa kabarmu?
- **Khalid:** أَنَا بِخَيْرٍ، وَأَنْتَ؟ -> Saya baik, dan kamu?
- **Karim:** أَنَا بِخَيْرٍ. كَانَ الدَّرْسُ جَيِّدًا. -> Saya baik. Pelajarannya bagus.
- **Khalid:** نَعَمْ، وَسَأُرَاجِعُ الْكَلِمَاتِ الْيَوْمَ. -> Ya, dan saya akan mengulang kosakata hari ini.
- **Karim:** جَيِّدٌ. إِلَى اللِّقَاءِ غَدًا. -> Bagus. Sampai bertemu besok.
