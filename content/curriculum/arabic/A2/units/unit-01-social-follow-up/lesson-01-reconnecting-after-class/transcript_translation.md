# Transcript Translation

- **Khalid:** السَّلَامُ عَلَيْكِ يَا مَرْيَمُ. -> Assalamu'alaikum, Maryam.
- **Maryam:** وَعَلَيْكُمُ السَّلَامُ يَا خَالِدُ. كَيْفَ حَالُكَ؟ -> Wa'alaikumussalam, Khalid. Apa kabarmu?
- **Khalid:** أَنَا بِخَيْرٍ، وَأَنْتِ؟ -> Saya baik, dan kamu?
- **Maryam:** أَنَا بِخَيْرٍ. كَانَ الدَّرْسُ جَيِّدًا. -> Saya baik. Pelajarannya bagus.
- **Khalid:** نَعَمْ، وَسَأُرَاجِعُ الْكَلِمَاتِ الْيَوْمَ. -> Ya, dan saya akan mengulang kosakata hari ini.
- **Maryam:** جَيِّدٌ. إِلَى اللِّقَاءِ غَدًا. -> Bagus. Sampai bertemu besok.
