# Transcript Translation

- **Rami:** مَاذَا تَفْعَلُ بَعْدَ الدَّرْسِ؟ -> Apa yang kamu lakukan setelah pelajaran?
- **Zayd:** سَأَذْهَبُ إِلَى الْمَكْتَبَةِ. -> Saya akan pergi ke perpustakaan.
- **Rami:** لِمَاذَا؟ -> Mengapa?
- **Zayd:** لِأَنَّنِي أُرِيدُ أَنْ أَتَدَرَّبَ عَلَى الْقِرَاءَةِ. -> Karena saya ingin berlatih membaca.
- **Rami:** هَذَا جَيِّدٌ. أَنَا سَأَكْتُبُ الْوَاجِبَ. -> Itu bagus. Saya akan menulis tugas.
- **Zayd:** جَمِيلٌ. وَهَلْ تَحْتَاجُ إِلَى مُسَاعَدَةٍ؟ -> Bagus. Apakah kamu butuh bantuan?
- **Rami:** لَا، شُكْرًا. الْوَاجِبُ سَهْلٌ. -> Tidak, terima kasih. Tugasnya mudah.
