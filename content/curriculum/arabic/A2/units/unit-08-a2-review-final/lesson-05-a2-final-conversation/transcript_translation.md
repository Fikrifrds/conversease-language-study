# Transcript Translation

- **Maryam:** كَيْفَ كَانَتْ تَجْرِبَتُكَ فِي الْمَدِينَةِ يَا خَالِدُ؟ -> Bagaimana pengalamanmu di kota, Khalid?
- **Khalid:** كَانَتْ جَيِّدَةً. رَأَيْتُ مَكَانًا جَدِيدًا وَاشْتَرَيْتُ كِتَابًا. -> Pengalamannya baik. Saya melihat tempat baru dan membeli buku.
- **Maryam:** هَلْ تُرِيدُ أَنْ تَذْهَبَ مَعِي إِلَى الْمَرْكَزِ غَدًا؟ -> Apakah kamu ingin pergi bersama saya ke pusat besok?
- **Khalid:** نَعَمْ، وَلَكِنْ كَيْفَ أَذْهَبُ إِلَى الْمَرْكَزِ؟ -> Ya, tetapi bagaimana saya pergi ke pusat?
- **Maryam:** اِمْشِ مُسْتَقِيمًا، ثُمَّ اِتَّجِهْ يَسَارًا عِنْدَ الْمَكْتَبَةِ. -> Berjalan lurus, lalu belok kiri di perpustakaan.
- **Khalid:** جَيِّدٌ. أُرِيدُ أَيْضًا أَنْ أُؤَكِّدَ مَوْعِدِي فِي الْعِيَادَةِ. -> Bagus. Saya juga ingin mengonfirmasi janji saya di klinik.
- **Maryam:** اِتَّصِلْ بِالِاسْتِقْبَالِ وَقُلْ: أُرِيدُ أَنْ أُؤَكِّدَ الْمَوْعِدَ. -> Hubungi penerimaan dan katakan: saya ingin mengonfirmasi janji.
- **Khalid:** شُكْرًا. مَا رَأْيُكِ فِي خُطَّتِنَا؟ -> Terima kasih. Apa pendapatmu tentang rencana kita?
- **Maryam:** أَعْتَقِدُ أَنَّ الْخُطَّةَ جَيِّدَةٌ لِأَنَّهَا وَاضِحَةٌ. -> Saya pikir rencananya baik karena jelas.
- **Khalid:** أُوَافِقُكِ. نَلْتَقِي غَدًا السَّاعَةَ الْعَاشِرَةَ. -> Saya setuju denganmu. Kita bertemu besok jam sepuluh.
