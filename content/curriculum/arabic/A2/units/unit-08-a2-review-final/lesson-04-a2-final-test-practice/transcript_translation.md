# Transcript Translation

- **Hakim:** مَاذَا فَعَلْتَ أَمْسِ يَا رَامِي؟ -> Apa yang kamu lakukan kemarin, Rami?
- **Rami:** ذَهَبْتُ إِلَى الْمَكْتَبَةِ، ثُمَّ دَرَسْتُ بَعْضَ الْعَرَبِيَّةِ. -> Saya pergi ke perpustakaan, lalu belajar sedikit bahasa Arab.
- **Hakim:** أَيْنَ تُفَضِّلُ أَنْ تَدْرُسَ؟ -> Di mana kamu lebih suka belajar?
- **Rami:** أُفَضِّلُ الْمَكْتَبَةَ لِأَنَّهَا هَادِئَةٌ. -> Saya lebih memilih perpustakaan karena tenang.
- **Hakim:** كَيْفَ تَطْلُبُ مَوْعِدًا؟ -> Bagaimana kamu meminta janji?
- **Rami:** أَقُولُ: أُرِيدُ مَوْعِدًا مَعَ الطَّبِيبِ. -> Saya mengatakan: saya ingin janji dengan dokter.
- **Hakim:** جَيِّدٌ. مَا رَأْيُكَ فِي الدِّرَاسَةِ كُلَّ يَوْمٍ؟ -> Bagus. Apa pendapatmu tentang belajar setiap hari?
- **Rami:** فِي رَأْيِي، هَذَا مُفِيدٌ، وَلَكِنَّهُ يَحْتَاجُ إِلَى وَقْتٍ. -> Menurut saya, itu bermanfaat, tetapi membutuhkan waktu.
