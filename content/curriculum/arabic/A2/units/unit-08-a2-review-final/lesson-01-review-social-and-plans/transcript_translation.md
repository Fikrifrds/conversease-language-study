# Transcript Translation

- **Khalid:** كَيْفَ كَانَ دَرْسُكِ الْيَوْمَ يَا مَرْيَمُ؟ -> Bagaimana pelajaranmu hari ini, Maryam?
- **Maryam:** كَانَ مُفِيدًا، وَلَكِنَّهُ كَانَ طَوِيلًا قَلِيلًا. -> Pelajarannya bermanfaat, tetapi sedikit panjang.
- **Khalid:** مَاذَا فَعَلْتِ بَعْدَ الدَّرْسِ؟ -> Apa yang kamu lakukan setelah pelajaran?
- **Maryam:** قَرَأْتُ النَّصَّ مَرَّةً أُخْرَى. -> Saya membaca teks sekali lagi.
- **Khalid:** هَلْ تُرِيدِينَ أَنْ تَدْرُسِي مَعِي مَسَاءً؟ -> Apakah kamu ingin belajar bersama saya sore ini?
- **Maryam:** نَعَمْ. مَتَى نَلْتَقِي؟ -> Ya. Kapan kita bertemu?
- **Khalid:** نَلْتَقِي السَّاعَةَ الْخَامِسَةَ أَمَامَ الْمَرْكَزِ. -> Kita bertemu jam lima di depan pusat.
- **Maryam:** حَسَنًا، الْوَقْتُ مُنَاسِبٌ. -> Baik, waktunya cocok.
