# Transcript Translation

- **Ahmad:** كَيْفَ كَانَتْ رِحْلَتُكَ إِلَى الْحَدِيقَةِ؟ -> Bagaimana perjalananmu ke taman?
- **Rami:** كَانَتِ التَّجْرِبَةُ جَيِّدَةً. -> Pengalamannya baik.
- **Ahmad:** مَاذَا رَأَيْتَ هُنَاكَ؟ -> Apa yang kamu lihat di sana?
- **Rami:** رَأَيْتُ مَكَانًا جَمِيلًا وَبُحَيْرَةً صَغِيرَةً. -> Saya melihat tempat indah dan danau kecil.
- **Ahmad:** هَلْ أَعْجَبَكَ الطَّعَامُ؟ -> Apakah kamu suka makanannya?
- **Rami:** نَعَمْ، أَعْجَبَنِي الطَّعَامُ، وَلَكِنَّ الْمَكَانَ كَانَ مُزْدَحِمًا. -> Ya, saya suka makanannya, tetapi tempatnya ramai.
- **Ahmad:** هَلْ تُرِيدُ أَنْ تَذْهَبَ مَرَّةً أُخْرَى؟ -> Apakah kamu ingin pergi sekali lagi?
- **Rami:** نَعَمْ، أُرِيدُ ذَلِكَ. -> Ya, saya ingin itu.
