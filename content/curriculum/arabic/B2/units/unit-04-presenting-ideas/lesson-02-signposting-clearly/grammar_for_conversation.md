# Pola Percakapan

Gunakan أَوَّلًا، ثُمَّ نَنْتَقِلُ إِلَى، dan أَخِيرًا untuk memandu presentasi.

```txt
أَوَّلًا دَعِينِي أَشْرَحُ الْمُشْكِلَةَ ...
ثُمَّ نَنْتَقِلُ إِلَى الْفَائِدَةِ ...
بَعْدَ ذَلِكَ نَنْظُرُ إِلَى ...
هُنَا نُقْطَةٌ مُهِمَّةٌ يَجِبُ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
