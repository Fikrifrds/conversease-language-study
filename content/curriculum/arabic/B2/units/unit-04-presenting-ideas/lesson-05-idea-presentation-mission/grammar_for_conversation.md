# Pola Percakapan

Gunakan لَدَيَّ فِكْرَةٌ، الدَّلِيلُ هُوَ، dan أَطْلُبُ مُوَافَقَتَكِ untuk pitch singkat.

```txt
لَدَيَّ فِكْرَةٌ تُوَفِّرُ وَقْتَ ...
الدَّلِيلُ هُوَ تَجْرِبَةٌ نَجَحَتْ ...
الْخَطَرُ مَحْدُودٌ لِأَنَّنَا نَبْدَأُ ...
لَنْ يَأْخُذَ هَذَا أَكْثَرَ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
