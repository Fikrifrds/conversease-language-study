# Pola Percakapan

Gunakan مِنْ نَاحِيَةٍ ... وَمِنْ نَاحِيَةٍ أُخْرَى، dan الْمُقَايَضَةُ هُنَا untuk membahas tradeoff.

```txt
الْمُقَايَضَةُ هُنَا بَيْنَ السُّرْعَةِ ...
مِنْ نَاحِيَةٍ السُّرْعَةُ تُعْطِينَا ...
وَمِنْ نَاحِيَةٍ أُخْرَى تُهَدِّدُ ...
الْقَرَارُ يَعْتَمِدُ عَلَى خَطَرِ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
