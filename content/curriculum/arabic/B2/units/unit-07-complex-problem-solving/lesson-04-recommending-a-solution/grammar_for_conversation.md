# Pola Percakapan

Gunakan أُوصِي بِـ، نُجَرِّبُهُ أُسْبُوعًا، dan نَقِيسُ النَّتِيجَةَ untuk merekomendasikan solusi.

```txt
أُوصِي بِقَائِمَةِ فَحْصٍ قَصِيرَةٍ ...
هَذَا الْحَلُّ بَسِيطٌ وَقَابِلٌ ...
نُجَرِّبُهُ أُسْبُوعًا ثُمَّ نَقِيسُ ...
إِنْ لَمْ يَنْفَعْ نُعَدِّلُهُ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
