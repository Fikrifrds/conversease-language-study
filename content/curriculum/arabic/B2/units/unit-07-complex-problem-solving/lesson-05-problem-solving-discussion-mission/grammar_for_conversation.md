# Pola Percakapan

Gabungkan أَبْدَأُ بِتَعْرِيفِ، أَسْأَلُ عَنِ السَّبَبِ، dan نَخْرُجُ بِخُطْوَةٍ untuk diskusi pemecahan masalah utuh.

```txt
أَبْدَأُ بِتَعْرِيفِ الْمُشْكِلَةِ لَا ...
ثُمَّ أَسْأَلُ عَنِ السَّبَبِ ...
أُذَكِّرُ الْفَرِيقَ أَلَّا يَقْفِزَ ...
أُرَتِّبُ النِّقَاشَ حَوْلَ مِعْيَارٍ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
