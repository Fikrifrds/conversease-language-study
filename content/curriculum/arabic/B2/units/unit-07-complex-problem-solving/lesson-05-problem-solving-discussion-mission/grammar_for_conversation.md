# Pola Percakapan

Gunakan أَرَى أَنَّ، قَدْ يَكُونُ، لِكُلِّ، dan أُوصِي بِـ untuk problem solving B2.

```txt
أَرَى أَنَّ ... جُزْءٌ مِنْ ...
السَّبَبُ ... قَدْ يَكُونُ ...
لِكُلِّ حَلٍّ ...
أُوصِي بِـ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
