# Pola Percakapan

Gunakan أَرَى أَنَّ ... جُزْءٌ مِنْ، لَا نَبْدَأُ بِالْحَلِّ، dan نُحَدِّدُ السُّؤَالَ untuk membingkai masalah.

```txt
أَرَى أَنَّ هَذِهِ النُّقْطَةَ ...
لَا نَبْدَأُ بِالْحَلِّ قَبْلَ ...
نُحَدِّدُ السُّؤَالَ الصَّحِيحَ أَوَّلًا ...
الْأَعْرَاضُ كَثِيرَةٌ لَكِنَّ السَّبَبَ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
