# Pola Percakapan

Gunakan السَّبَبُ الْجَذْرِيُّ، يَعُودُ ذَلِكَ إِلَى، dan لَيْسَ مُجَرَّدَ عَرَضٍ untuk menjelaskan penyebab.

```txt
لِنُفَرِّقْ بَيْنَ الْعَرَضِ وَالسَّبَبِ ...
يَعُودُ التَّأَخُّرُ إِلَى ضَعْفِ ...
هَذَا لَيْسَ مُجَرَّدَ نَقْصٍ ...
الدَّلِيلُ أَنَّ الْمُشْكِلَةَ تَتَكَرَّرُ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
