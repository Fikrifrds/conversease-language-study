# Pola Percakapan

Gunakan مِثَالٌ مَلْمُوسٌ، حَدَثَ هَذَا فِعْلًا، dan هَذَا الْمِثَالُ يُظْهِرُ untuk mendukung argumen dengan contoh.

```txt
لِأُوَضِّحَ فِكْرَتِي سَأُعْطِي مِثَالًا ...
مِثَالٌ وَاحِدٌ مَلْمُوسٌ يُسَاوِي ...
حَدَثَ هَذَا فِعْلًا مَعَ ...
هَذَا الْمِثَالُ يُظْهِرُ الْمُشْكِلَةَ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
