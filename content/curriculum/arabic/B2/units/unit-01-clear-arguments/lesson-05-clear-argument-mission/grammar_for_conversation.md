# Pola Percakapan

Gabungkan مَوْقِفِي، السَّبَبُ، مِثَالُ ذَلِكَ، أَفْهَمُ ... لَكِنَّ، dan أُوصِي بِـ untuk argumen B2 yang lengkap.

```txt
مَوْقِفِي أَنَّ الْمُورِّدَ الثَّانِيَ ...
السَّبَبُ الْأَوَّلُ هُوَ سُرْعَةُ ...
مِثَالُ ذَلِكَ أَنَّهُ رَدَّ ...
أَفْهَمُ أَنَّهُ أَغْلَى لَكِنَّ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
