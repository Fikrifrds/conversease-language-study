# Pola Percakapan

Gunakan مَوْقِفِي هُوَ، أَدْعَمُ، مِثَالُ ذَلِكَ، dan وَلَكِنَّ untuk membuat argumen B2 yang terstruktur.

```txt
مَوْقِفِي هُوَ أَنَّ ...
أَدْعَمُ هَذَا الرَّأْيَ بِـ ...
مِثَالُ ذَلِكَ أَنَّ ...
أَفْهَمُ ... وَلَكِنَّ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
