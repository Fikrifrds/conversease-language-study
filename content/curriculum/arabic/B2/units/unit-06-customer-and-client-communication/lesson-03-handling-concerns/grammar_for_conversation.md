# Pola Percakapan

Gunakan أَعْتَرِفُ بِقَلَقِكَ، السَّبَبُ هُوَ، dan إِلَيْكَ الْحَلُّ untuk menangani concern.

```txt
أَعْتَرِفُ بِقَلَقِكَ قَبْلَ أَيِّ ...
التَّأْخِيرُ جَاءَ مِنْ مُرَاجَعَةِ ...
أَفْهَمُ هَذَا الِاهْتِمَامَ وَسَأُوَضِّحُ ...
إِلَيْكَ الْحَلُّ: مَوْعِدٌ مُحَافِظٌ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
