# Pola Percakapan

Gunakan أُرِيدُ أَنْ أَفْهَمَ، لَدَيْنَا، أَفْهَمُ، dan سَأُرْسِلُ untuk komunikasi klien.

```txt
أُرِيدُ أَنْ أَفْهَمَ ...
لَدَيْنَا خِيَارَانِ ...
أَفْهَمُ ... وَسَأُوَضِّحُ ...
الْخُطْوَةُ التَّالِيَةُ هِيَ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
