# Pola Percakapan

Gunakan لَدَيْنَا خِيَارَانِ، الْأَوَّلُ ... وَالثَّانِي، dan أَيُّهُمَا يُنَاسِبُكَ untuk menjelaskan opsi.

```txt
لَدَيْنَا خِيَارَانِ وَكُلُّ خِيَارٍ ...
الْأَوَّلُ أَسْرَعُ وَالثَّانِي أَشْمَلُ ...
أُوَضِّحُ التَّكْلِفَةَ وَمَا يَحْصُلُ ...
أَيُّهُمَا يُنَاسِبُ حَاجَتَكَ أَكْثَرَ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
