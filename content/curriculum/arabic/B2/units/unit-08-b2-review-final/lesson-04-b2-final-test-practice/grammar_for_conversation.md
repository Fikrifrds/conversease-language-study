# Pola Percakapan

Gunakan مَوْقِفِي وَاضِحٌ، السَّبَبُ الْأَوَّلُ ... وَالثَّانِي، dan خُلَاصَتِي untuk menjawab ujian.

```txt
مَوْقِفِي وَاضِحٌ وَسَأَشْرَحُهُ بِنُقْطَتَيْنِ ...
السَّبَبُ الْأَوَّلُ هُوَ كَذَا ...
مِثَالٌ بَسِيطٌ عَلَى ذَلِكَ ...
أَفْهَمُ السُّؤَالَ وَلَوْ نَسِيتُ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
