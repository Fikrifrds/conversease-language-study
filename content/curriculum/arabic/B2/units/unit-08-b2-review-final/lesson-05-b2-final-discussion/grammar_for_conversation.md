# Pola Percakapan

Gunakan أَرَى أَنَّ، الدَّلِيلُ هُوَ، dan أَقْتَرِحُ كَخُطْوَةٍ untuk diskusi B2 utuh.

```txt
أَرَى أَنَّ الْعَمَلَ الْمَرِنَ ...
الدَّلِيلُ هُوَ أَنَّ الْإِنْتَاجَ ...
أَفْهَمُ مَنْ يَخْشَى عَلَى ...
لَكِنَّ مِعْيَارَ الْحُضُورِ يَحُلُّ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
