# Pola Percakapan

Gabungkan أَبْدَأُ بِالرَّقْمِ، مَا لَا تَقُولُهُ الْبَيَانَاتُ، dan مَا الَّذِي نُغَيِّرُهُ untuk diskusi data.

```txt
أَبْدَأُ بِالرَّقْمِ الْأَقْرَبِ إِلَى ...
ثُمَّ أَذْكُرُ مَا لَا ...
النِّسْبَةُ قَدْ تُفْهَمُ خَطَأً ...
الرَّقْمُ مُؤَشِّرٌ لَا دَلِيلٌ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
