# Transcript Translation

- **Yusuf:** وَصَلَتْنَا مَعْلُومَاتٌ جَدِيدَةٌ عَنِ الْمُنَافِسِ. -> Kita dapat info baru soal kompetitor.
- **Hasan:** هَذِهِ الْمَعْلُومَةُ تُغَيِّرُ الصُّورَةَ قَلِيلًا. -> Informasi ini sedikit mengubah gambaran.
- **Yusuf:** كَيْفَ؟ -> Gimana?
- **Hasan:** كُنَّا نَظُنُّ أَنَّ السِّعْرَ هُوَ السَّبَبُ، لَكِنْ قَبْلَ أَنْ نُقَرِّرَ، نَتَأَكَّدُ مِنَ الْمَصْدَرِ. -> Tadinya kita kira harga penyebabnya, tapi sebelum memutuskan, kita pastikan sumbernya.
- **Yusuf:** هَلْ نُغَيِّرُ الْخُطَّةَ كُلَّهَا؟ -> Apa kita ubah seluruh rencana?
- **Hasan:** أُعِيدُ النَّظَرَ، لَكِنْ لَا أُغَيِّرُ كُلَّ شَيْءٍ. -> Aku tinjau ulang, tapi tidak mengubah semuanya.
- **Yusuf:** فَمَا الْقَرَارُ؟ -> Jadi keputusannya?
- **Hasan:** لِذَلِكَ أُعِيدُ تَرْتِيبَ التَّوْصِيَةِ بِنَاءً عَلَيْهَا. -> Karena itu aku susun ulang rekomendasi berdasarkannya.
