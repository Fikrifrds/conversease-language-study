# Pola Percakapan

Gunakan مَنْ كَتَبَ، مَتَى نُشِرَ، dan هَلْ يُوجَدُ دَلِيلٌ untuk menilai keandalan sumber.

```txt
قَبْلَ أَنْ نَثِقَ مَنْ ...
تَارِيخُ النَّشْرِ مُهِمٌّ فَالْخَبَرُ ...
هَلْ يُوجَدُ دَلِيلٌ أَمْ ...
الْمَوْقِعُ غَيْرُ مَعْرُوفٍ فَلْنَكُنْ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
