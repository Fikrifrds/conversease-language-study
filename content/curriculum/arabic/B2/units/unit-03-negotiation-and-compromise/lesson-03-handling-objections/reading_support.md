# Bantuan Membaca

Baca dialog sebagai rangkaian argumen. Cari posisi utama, alasan, contoh, respons terhadap keberatan, lalu rekomendasi akhir.

- أَوْلَوِيَّتِي هِيَ أَنْ يَبْقَى التَّعَامُلُ مَعَ الِاعْتِرَاضَاتِ وَاضِحًا. -> Prioritas saya adalah agar menangani keberatan tetap jelas.
- أُقَدِّمُ الِاقْتِرَاحَ التَّالِيَ كَحَلٍّ عَمَلِيٍّ. -> Saya mengajukan usulan berikut sebagai solusi praktis.
- أَفْهَمُ قَلَقَكَ، وَلَكِنْ لَدَيْنَا خِيَارٌ آخَرُ. -> Saya memahami kekhawatiranmu, tetapi kita punya pilihan lain.
- يُمْكِنُنَا أَنْ نَصِلَ إِلَى حَلٍّ وَسَطٍ. -> Kita bisa mencapai solusi tengah.
- إِذَا اتَّفَقْنَا عَلَى ذَلِكَ، فَالْخُطْوَةُ التَّالِيَةُ وَاضِحَةٌ. -> Jika kita sepakat tentang itu, langkah berikutnya jelas.
