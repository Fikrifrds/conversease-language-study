# Bantuan Membaca

Baca kalimat dari kanan ke kiri. Cari dulu frasa penghubung, lalu pahami fungsi kalimat: konteks, alasan, hasil, atau langkah berikutnya.

- أُرِيدُ أَنْ أُوَضِّحَ مُشْكِلَةً صَغِيرَةً. -> Saya ingin menjelaskan masalah kecil.
- الْغُرْفَةُ غَيْرُ نَظِيفَةٍ تَمَامًا. -> Kamarnya tidak sepenuhnya bersih.
- هَلْ يُمْكِنُ أَنْ تُرْسِلُوا أَحَدًا؟ -> Bisakah kalian mengirim seseorang?
- أُقَدِّرُ مُسَاعَدَتَكُمْ. -> Saya menghargai bantuan kalian.
- إِذَا أَمْكَنَ، أُرِيدُ حَلًّا الْيَوْمَ. -> Jika memungkinkan, saya ingin solusi hari ini.
