# Transcript Translation

- **Ahmad:** كَيْفَ تَعَامَلْتَ مَعَ مُشْكِلَةِ الْغُرْفَةِ؟ -> Gimana kamu menangani masalah kamarnya?
- **Karim:** قُلْتُ بِأَدَبٍ: أُرِيدُ أَنْ أُوَضِّحَ مُشْكِلَةً صَغِيرَةً. -> Aku bilang dengan sopan: aku ingin menjelaskan masalah kecil.
- **Ahmad:** وَمَا الْمُشْكِلَةُ؟ -> Apa masalahnya?
- **Karim:** الْغُرْفَةُ غَيْرُ نَظِيفَةٍ تَمَامًا، فَسَأَلْتُ: هَلْ يُمْكِنُ أَنْ تُرْسِلُوا أَحَدًا؟ -> Kamarnya tidak sepenuhnya bersih, jadi aku tanya: bisakah kalian mengirim seseorang?
- **Ahmad:** وَهَلْ سَاعَدُوكَ؟ -> Apa mereka membantumu?
- **Karim:** أُقَدِّرُ مُسَاعَدَتَهُمْ. قُلْتُ: إِذَا أَمْكَنَ، أُرِيدُ حَلًّا الْيَوْمَ. -> Aku menghargai bantuan mereka. Aku bilang: kalau memungkinkan, aku ingin selesai hari ini.
