# Transcript Translation

- **Maryam:** هَلْ تُرِيدِينَ أَنْ تُضِيفِي شَيْئًا؟ -> Apakah kamu ingin menambahkan sesuatu?
- **Noura:** أُرِيدُ أَنْ أُضِيفَ نُقْطَةً صَغِيرَةً. -> Saya ingin menambahkan poin kecil.
- **Maryam:** هَلْ يُمْكِنُ أَنْ تُوَضِّحَ أَكْثَرَ؟ -> Bisakah kamu menjelaskan lebih banyak?
- **Noura:** أَقْتَرِحُ أَنْ نُرَاجِعَ الْخُطَّةَ أَوَّلًا. هَذَا يُمْكِنُ أَنْ يُقَلِّلَ الْخَطَأَ. -> Saya menyarankan kita meninjau rencana dulu. Ini bisa mengurangi kesalahan.
- **Maryam:** مَا الْخُطْوَةُ التَّالِيَةُ بَعْدَ ذَلِكَ؟ -> Apa langkah berikutnya setelah itu?
- **Noura:** سَأَكْتُبُ النِّقَاطَ الْمُهِمَّةَ. -> Saya akan menulis poin-poin penting.
