# Bantuan Membaca

Baca kalimat dari kanan ke kiri. Cari dulu frasa penghubung, lalu pahami fungsi kalimat: konteks, alasan, hasil, atau langkah berikutnya.

- أَعْمَلُ عَلَى تَنْظِيمِ الْمَلَفَّاتِ. -> Saya mengerjakan pengaturan file.
- هَلْ تَقْصِدِينَ الْمَلَفَّاتِ الْجَدِيدَةَ فَقَطْ؟ -> Apakah maksudmu hanya file baru?
- أَنْجَزْتُ نِصْفَ الْعَمَلِ حَتَّى الْآنَ. -> Saya sudah menyelesaikan setengah pekerjaan sampai sekarang.
- سَأُرْسِلُ مُلَخَّصًا قَبْلَ الْمَسَاءِ. -> Saya akan mengirim ringkasan sebelum sore.
- إِذَا ظَهَرَتْ مُشْكِلَةٌ، سَأُخْبِرُكِ. -> Jika muncul masalah, saya akan memberitahumu.
