# Transcript Translation

- **Maryam:** أَعْطِينِي تَحْدِيثًا عَنِ الْعَمَلِ. -> Berikan saya update tentang pekerjaan.
- **Noura:** أَعْمَلُ عَلَى تَنْظِيمِ الْمَلَفَّاتِ. -> Saya mengerjakan pengaturan file.
- **Maryam:** مَا الْخُطْوَةُ التَّالِيَةُ بَعْدَ ذَلِكَ؟ -> Apa langkah berikutnya setelah itu?
- **Noura:** هَلْ تَقْصِدِينَ الْمَلَفَّاتِ الْجَدِيدَةَ فَقَطْ؟ أَنْجَزْتُ نِصْفَ الْعَمَلِ حَتَّى الْآنَ. -> Apakah maksudmu hanya file baru? Saya sudah menyelesaikan setengah pekerjaan sampai sekarang.
- **Maryam:** سَأُرْسِلُ مُلَخَّصًا قَبْلَ الْمَسَاءِ. -> Saya akan mengirim ringkasan sebelum sore.
- **Noura:** إِذَا ظَهَرَتْ مُشْكِلَةٌ، سَأُخْبِرُكِ. -> Jika muncul masalah, saya akan memberitahumu.
