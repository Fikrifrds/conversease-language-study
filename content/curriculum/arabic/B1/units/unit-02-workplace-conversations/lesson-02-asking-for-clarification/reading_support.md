# Bantuan Membaca

Baca kalimat dari kanan ke kiri. Cari dulu frasa penghubung, lalu pahami fungsi kalimat: konteks, alasan, hasil, atau langkah berikutnya.

- هَلْ تَقْصِدِينَ أَنَّ الْمَوْعِدَ تَغَيَّرَ؟ -> Apakah maksudmu jadwalnya berubah?
- مَا الْمَقْصُودُ بِهَذِهِ النُّقْطَةِ؟ -> Apa maksud poin ini?
- أَسْأَلُ لِكَيْ أَتَأَكَّدَ فَقَطْ. -> Saya bertanya hanya untuk memastikan.
- هَلْ يُمْكِنُ أَنْ تُعِيدِي الشَّرْحَ بِاخْتِصَارٍ؟ -> Bisakah kamu mengulang penjelasan dengan singkat?
- الْآنَ فَهِمْتُ الْمَطْلُوبَ. -> Sekarang saya memahami yang diminta.
