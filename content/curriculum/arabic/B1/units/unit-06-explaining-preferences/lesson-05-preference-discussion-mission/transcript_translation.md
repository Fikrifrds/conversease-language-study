# Transcript Translation

- **Fatimah:** كَيْفَ تُنْهِي الْحِوَارَ بِخُطْوَةٍ تَالِيَةٍ؟ -> Bagaimana kamu mengakhiri percakapan dengan langkah berikutnya?
- **Noura:** سَأُقَارِنُ بَيْنَ خِيَارَيْنِ. -> Saya akan membandingkan dua pilihan.
- **Fatimah:** مَا الْخُطْوَةُ التَّالِيَةُ بَعْدَ ذَلِكَ؟ -> Apa langkah berikutnya setelah itu?
- **Noura:** أُفَضِّلُ الْخِيَارَ الْأَوَّلَ لِأَنَّهُ أَسْهَلُ. لَكِنَّ الْخِيَارَ الثَّانِي أَقْوَى. -> Saya lebih memilih pilihan pertama karena lebih mudah. Tetapi pilihan kedua lebih kuat.
- **Fatimah:** نَحْتَاجُ إِلَى تَوَازُنٍ بَيْنَ السُّرْعَةِ وَالْجَوْدَةِ. -> Kita perlu keseimbangan antara kecepatan dan kualitas.
- **Noura:** إِذَنْ، نَخْتَارُ الْحَلَّ الْمُتَوَازِنَ. -> Kalau begitu, kita pilih solusi yang seimbang.
