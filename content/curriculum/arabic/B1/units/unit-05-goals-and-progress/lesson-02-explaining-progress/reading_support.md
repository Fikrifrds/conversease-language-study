# Bantuan Membaca

Baca kalimat dari kanan ke kiri. Cari dulu frasa penghubung, lalu pahami fungsi kalimat: konteks, alasan, hasil, atau langkah berikutnya.

- تَقَدَّمْتُ قَلِيلًا هَذَا الشَّهْرَ. -> Saya sedikit maju bulan ini.
- أَصْبَحْتُ أَفْهَمُ الْحِوَارَ أَسْرَعَ. -> Saya menjadi lebih cepat memahami dialog.
- مَا زِلْتُ أُخْطِئُ فِي بَعْضِ الْجُمَلِ. -> Saya masih salah pada beberapa kalimat.
- لَكِنَّ التَّحَسُّنَ وَاضِحٌ. -> Tetapi perbaikannya jelas.
- سَأُوَاصِلُ التَّدْرِيبَ. -> Saya akan terus berlatih.
