# Transcript Translation

- **Sara:** مَا النُّقْطَةُ الْأَسَاسِيَّةُ فِي هَذَا الْمَوْضُوعِ؟ -> Apa poin utama dalam topik ini?
- **Layla:** تَقَدَّمْتُ قَلِيلًا هَذَا الشَّهْرَ. -> Saya sedikit maju bulan ini.
- **Sara:** هَلْ يُمْكِنُ أَنْ تُوَضِّحَ أَكْثَرَ؟ -> Bisakah kamu menjelaskan lebih banyak?
- **Layla:** أَصْبَحْتُ أَفْهَمُ الْحِوَارَ أَسْرَعَ. مَا زِلْتُ أُخْطِئُ فِي بَعْضِ الْجُمَلِ. -> Saya menjadi lebih cepat memahami dialog. Saya masih salah pada beberapa kalimat.
- **Sara:** لَكِنَّ التَّحَسُّنَ وَاضِحٌ. -> Tetapi perbaikannya jelas.
- **Layla:** سَأُوَاصِلُ التَّدْرِيبَ. -> Saya akan terus berlatih.
