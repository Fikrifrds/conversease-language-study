# Transcript Translation

- **Sara:** مَا الْفِكْرَةُ الرَّئِيسِيَّةُ الَّتِي تُرِيدُ بَدْءَ الْحِوَارِ بِهَا؟ -> Ide utama apa yang ingin kamu pakai untuk memulai percakapan?
- **Layla:** هَدَفِي هُوَ تَحْسِينُ الْمُحَادَثَةِ. -> Tujuan saya adalah memperbaiki percakapan.
- **Sara:** مَا التَّفْصِيلُ الْأَهَمُّ فِي هَذِهِ النُّقْطَةِ؟ -> Detail paling penting apa dalam poin ini?
- **Layla:** أُرِيدُ أَنْ أَتَكَلَّمَ بِثِقَةٍ أَكْبَرَ. لِذَلِكَ أَتَدَرَّبُ كُلَّ يَوْمٍ. -> Saya ingin berbicara dengan lebih percaya diri. Karena itu saya berlatih setiap hari.
- **Sara:** أَقِيسُ التَّقَدُّمَ بِالْمُرَاجَعَةِ. -> Saya mengukur kemajuan dengan peninjauan ulang.
- **Layla:** هَذَا الْهَدَفُ مُهِمٌّ لِي. -> Tujuan ini penting bagi saya.
