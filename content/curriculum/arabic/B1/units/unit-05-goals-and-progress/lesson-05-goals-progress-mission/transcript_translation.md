# Transcript Translation

- **Sara:** رَاجِعِي هَدَفَكِ وَتَقَدُّمَكِ بِاخْتِصَارٍ. -> Tinjau tujuan dan kemajuanmu secara singkat.
- **Layla:** هَدَفِي وَاضِحٌ لِهَذَا الشَّهْرِ. تَقَدَّمْتُ فِي الِاسْتِمَاعِ، وَلَكِنَّ الْكَلَامَ أَصْعَبُ. -> Tujuanku jelas untuk bulan ini. Aku maju dalam listening, tapi speaking lebih sulit.
- **Sara:** وَمَا أَكْبَرُ صُعُوبَةٍ؟ -> Apa kesulitan terbesarnya?
- **Layla:** أَكْبَرُ تَحَدٍّ هُوَ التَّرَدُّدُ. -> Tantangan terbesar adalah keraguan.
- **Sara:** فَمَا خُطَّتُكِ؟ -> Jadi apa rencanamu?
- **Layla:** سَأَتَدَرَّبُ عَلَى إِجَابَاتٍ قَصِيرَةٍ، وَسَأُرَاجِعُ النَّتِيجَةَ بَعْدَ أُسْبُوعٍ. -> Aku akan berlatih jawaban pendek, dan meninjau hasilnya setelah seminggu.
