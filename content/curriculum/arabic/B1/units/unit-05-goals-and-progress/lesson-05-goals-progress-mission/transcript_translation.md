# Transcript Translation

- **Sara:** كَيْفَ تُنْهِي الْحِوَارَ بِخُطْوَةٍ تَالِيَةٍ؟ -> Bagaimana kamu mengakhiri percakapan dengan langkah berikutnya?
- **Layla:** هَدَفِي وَاضِحٌ لِهَذَا الشَّهْرِ. -> Tujuan saya jelas untuk bulan ini.
- **Sara:** مَا الْخُطْوَةُ التَّالِيَةُ بَعْدَ ذَلِكَ؟ -> Apa langkah berikutnya setelah itu?
- **Layla:** تَقَدَّمْتُ فِي الِاسْتِمَاعِ، وَلَكِنَّ الْكَلَامَ أَصْعَبُ. أَكْبَرُ تَحَدٍّ هُوَ التَّرَدُّدُ. -> Saya maju dalam listening, tetapi speaking lebih sulit. Tantangan terbesar adalah keraguan.
- **Sara:** سَأَتَدَرَّبُ عَلَى إِجَابَاتٍ قَصِيرَةٍ. -> Saya akan berlatih jawaban pendek.
- **Layla:** سَأُرَاجِعُ النَّتِيجَةَ بَعْدَ أُسْبُوعٍ. -> Saya akan meninjau hasilnya setelah seminggu.
