# Pola Percakapan

Gunakan لِنَنْتَقِلْ مِنْ ... إِلَى، بِنَاءً عَلَى مَا سَبَقَ، dan قَبْلَ أَنْ نُكْمِلَ untuk transisi.

```txt
لِنَنْتَقِلْ مِنَ السَّبَبِ إِلَى ...
بِنَاءً عَلَى مَا سَبَقَ ...
قَبْلَ أَنْ نُكْمِلَ أُلَخِّصُ ...
هَذِهِ الْجُمْلَةُ تُهَيِّئُ الْجُمْهُورَ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
