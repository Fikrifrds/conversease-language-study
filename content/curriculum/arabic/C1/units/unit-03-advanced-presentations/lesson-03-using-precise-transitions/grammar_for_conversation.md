# Pola Percakapan

Gunakan حَتَّى، بَعْدَ ذَلِكَ، مِنْ ... إِلَى ...، dan إِذَا untuk presentasi C1.

```txt
سَأُؤَطِّرُ ... حَتَّى ...
سَأَبْنِي ... خُطْوَةً فَخُطْوَةً
نَنْتَقِلُ مِنْ ... إِلَى ...
إِذَا جَاءَ ... فَـ ... ثُمَّ ...
```

Gunakan pola ini untuk menjaga presisi. Jawaban C1 boleh kompleks, tetapi setiap klausa harus punya fungsi yang jelas.
