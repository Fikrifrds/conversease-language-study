# Pola Percakapan

Gunakan أَبْدَأُ مِنَ، يَقُودُنَا ذَلِكَ إِلَى، dan الْآنَ نَنْظُرُ إِلَى untuk membangun alur.

```txt
أَبْدَأُ مِنَ الْمُشْكِلَةِ قَبْلَ ...
يَقُودُنَا ذَلِكَ إِلَى الدَّلِيلِ ...
الْآنَ نَنْظُرُ إِلَى الْخِيَارَيْنِ ...
لَوْ قَاطَعَنِي أَحَدٌ أُجِيبُ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
