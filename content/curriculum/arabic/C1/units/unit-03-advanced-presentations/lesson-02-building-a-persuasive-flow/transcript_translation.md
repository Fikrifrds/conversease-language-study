# Transcript Translation

- **Salma:** كَيْفَ تُطَبِّقُ مَهَارَةَ بِنَاءِ تَسَلْسُلٍ إِقْنَاعِيٍّ فِي حِوَارٍ رَسْمِيٍّ؟ -> Bagaimana kamu menerapkan building a persuasive flow dalam percakapan formal?
- **Lina:** فِي مَهَارَةِ بِنَاءِ تَسَلْسُلٍ إِقْنَاعِيٍّ، سَأُؤَطِّرُ الْمَوْضُوعَ أَوَّلًا حَتَّى يَفْهَمَ الْجُمْهُورُ سِيَاقَهُ. -> Dalam skill building a persuasive flow, Saya akan membingkai topiknya dulu agar audiens memahami konteksnya.
- **Salma:** هَذَا جَيِّدٌ، وَلَكِنْ مَا الْجَانِبُ الَّذِي يَحْتَاجُ إِلَى دِقَّةٍ أَكْبَرَ؟ -> Bagus, tetapi sisi mana yang membutuhkan presisi lebih besar?
- **Lina:** عِنْدَ تَطْبِيقِ بِنَاءِ تَسَلْسُلٍ إِقْنَاعِيٍّ، بَعْدَ ذَلِكَ، سَأَبْنِي التَّسَلْسُلَ الْإِقْنَاعِيَّ خُطْوَةً فَخُطْوَةً. الِانْتِقَالُ الْأَدَقُّ هُنَا هُوَ أَنْ نَنْتَقِلَ مِنَ السَّبَبِ إِلَى التَّأْثِيرِ. -> Saat menerapkan building a persuasive flow, Setelah itu, saya akan membangun alur persuasif langkah demi langkah. Transisi yang lebih tepat di sini adalah berpindah dari sebab ke dampak.
- **Salma:** كَيْفَ تَتَعَامَلُ مَعَ النُّقْطَةِ الْمُعَارِضَةِ أَوْ سُوءِ الْفَهْمِ؟ -> Bagaimana kamu menangani counterpoint atau salah paham?
- **Lina:** إِذَا ظَهَرَ تَحَدٍّ فِي هَذِهِ الْمَهَارَةِ، إِذَا جَاءَ سُؤَالٌ صَعْبٌ، فَسَأَعْتَرِفُ بِحُدُودِ الْجَوَابِ ثُمَّ أُوَضِّحُ الْأَسَاسَ. -> Jika muncul tantangan dalam skill ini, Jika ada pertanyaan sulit, saya akan mengakui batas jawaban lalu menjelaskan dasarnya.
- **Salma:** مَا الصِّيَاغَةُ النِّهَائِيَّةُ الَّتِي تُنَاسِبُ مَوْقِفًا رَسْمِيًّا؟ -> Apa formulasi akhir yang cocok untuk situasi formal?
- **Lina:** فِي الصِّيَاغَةِ النِّهَائِيَّةِ لِهَذِهِ الْمَهَارَةِ، فِي النِّهَايَةِ، أُرِيدُ أَنْ يَخْرُجَ الْجُمْهُورُ بِرِسَالَةٍ وَاضِحَةٍ. -> Dalam final formulation untuk building a persuasive flow, Pada akhirnya, saya ingin audiens pulang dengan pesan yang jelas.
