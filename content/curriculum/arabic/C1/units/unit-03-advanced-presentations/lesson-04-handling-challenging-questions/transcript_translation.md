# Transcript Translation

- **Salma:** بَعْدَ الْعَرْضِ سَأَسْأَلُكِ سُؤَالًا صَعْبًا عَنِ التَّكْلِفَةِ. -> Setelah presentasi aku akan tanya pertanyaan sulit soal biaya.
- **Lina:** هَذِهِ نُقْطَةٌ حَقِيقِيَّةٌ، وَأَشْكُرُكِ عَلَيْهَا. -> Ini poin yang nyata, dan terima kasih atasnya.
- **Salma:** فَمَا تَكْلِفَتُهُ بِالضَّبْطِ؟ -> Jadi berapa persis biayanya?
- **Lina:** أُفَرِّقُ بَيْنَ تَكْلِفَةِ الْآنَ وَتَكْلِفَةِ التَّأْخِيرِ. لَا أَمْلِكُ الرَّقْمَ الدَّقِيقَ الْآنَ، لَكِنِّي سَأُؤَكِّدُهُ غَدًا. -> Aku bedakan biaya sekarang dan biaya keterlambatan. Aku tidak punya angka persisnya sekarang, tapi kupastikan besok.
- **Salma:** أَلَا يَبْدُو هَذَا تَهَرُّبًا؟ -> Bukankah itu terkesan menghindar?
- **Lina:** دَعِينِي أَرْبِطُ الْجَوَابَ بِهَدَفِ الْعَرْضِ. -> Biar kuhubungkan jawaban ke tujuan presentasi.
- **Salma:** جَيِّدٌ. -> Bagus.
- **Lina:** هَكَذَا أُجِيبُ بِصِدْقٍ دُونَ أَنْ أَفْقِدَ خَيْطَ الْعَرْضِ. -> Dengan begitu aku jawab jujur tanpa kehilangan alur.
