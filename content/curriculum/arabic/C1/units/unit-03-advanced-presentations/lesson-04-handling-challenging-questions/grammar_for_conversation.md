# Pola Percakapan

Gunakan هَذِهِ نُقْطَةٌ حَقِيقِيَّةٌ، أُفَرِّقُ بَيْنَ، dan لَا أَمْلِكُ الرَّقْمَ لَكِنْ untuk menjawab pertanyaan sulit.

```txt
هَذِهِ نُقْطَةٌ حَقِيقِيَّةٌ وَأَشْكُرُكِ ...
أُفَرِّقُ بَيْنَ تَكْلِفَةِ الْآنَ ...
لَا أَمْلِكُ الرَّقْمَ الْآنَ ...
دَعِينِي أَرْبِطُ الْجَوَابَ بِهَدَفِ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
