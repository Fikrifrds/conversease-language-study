# Pola Percakapan

Gunakan مَوْقِفِي دَقِيقٌ، الدَّلِيلُ، dan خُلَاصَتِي الدَّقِيقَةُ untuk jawaban ujian C1.

```txt
مَوْقِفِي دَقِيقٌ: أُؤَيِّدُ لَكِنْ ...
الدَّلِيلُ هُوَ نَتِيجَةٌ تَكَرَّرَتْ ...
مِنْ نَاحِيَةٍ أُخْرَى أَعْتَرِفُ ...
لَوْ نَسِيتُ مُصْطَلَحًا أَشْرَحُ ...
```

Pakai pola ini untuk menjaga jawaban tetap koheren. Jika respons terasa panjang, pisahkan menjadi dua kalimat tetapi tetap jaga hubungan logisnya.
