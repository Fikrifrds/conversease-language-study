# Transcript Translation

- **Khalid:** مَا الطَّرِيقَةُ الأَدَقُّ لِاسْتِخْدَامِ هَذِهِ الْمَهَارَةِ؟ -> Cara paling presisi apa untuk memakai skill ini?
- **Omar:** فِي مَهَارَةِ مُنَاقَشَةِ حُجَّةٍ بِأَدَبٍ، أَعْتَقِدُ أَنَّ الِافْتِرَاضَ الْأَسَاسِيَّ هُنَا غَيْرُ مَذْكُورٍ بِوُضُوحٍ. -> Dalam skill challenging an argument politely, Saya pikir asumsi dasarnya di sini belum disebutkan dengan jelas.
- **Khalid:** مَا الْفَرْقُ بَيْنَ الصِّيَاغَةِ الْمُبَاشِرَةِ وَالصِّيَاغَةِ الْأَدَقِّ هُنَا؟ -> Apa bedanya formulasi langsung dan formulasi yang lebih presisi dalam challenging an argument politely?
- **Omar:** عِنْدَ تَطْبِيقِ مُنَاقَشَةِ حُجَّةٍ بِأَدَبٍ، يُمْكِنُنَا أَنْ نُحَدِّيَ هَذِهِ الْحُجَّةَ مِنْ جِهَةِ الْأَدِلَّةِ. الدَّلِيلُ الْأَقْوَى هُوَ أَنَّ النَّتِيجَةَ تَكَرَّرَتْ فِي أَكْثَرَ مِنْ سِيَاقٍ. -> Saat menerapkan challenging an argument politely, Kita bisa menantang argumen ini dari sisi bukti. Bukti paling kuat adalah bahwa hasilnya berulang dalam lebih dari satu konteks.
- **Khalid:** كَيْفَ تُصَحِّحُ سُوءَ فَهْمٍ صَغِيرًا أَثْنَاءَ الْحِوَارِ؟ -> Bagaimana kamu memperbaiki salah paham kecil saat challenging an argument politely?
- **Omar:** إِذَا ظَهَرَ تَحَدٍّ فِي هَذِهِ الْمَهَارَةِ، تَحْتَ الضَّغْطِ، سَأُرَكِّزُ عَلَى النُّقْطَةِ الْمَرْكَزِيَّةِ وَلَا أُطِيلُ. -> Jika muncul tantangan dalam skill ini, Di bawah tekanan, saya akan fokus pada poin pusat dan tidak berpanjang-panjang.
- **Khalid:** مَا الْقَيْدُ الَّذِي سَتُضِيفُهُ لِجَعْلِ الْجَوَابِ أَكْثَرَ تَوَازُنًا؟ -> Batasan apa yang akan kamu tambahkan agar challenging an argument politely lebih seimbang?
- **Omar:** فِي الصِّيَاغَةِ النِّهَائِيَّةِ لِهَذِهِ الْمَهَارَةِ، لِذَلِكَ فَالرَّدُّ الْأَقْوَى هُوَ تَوْضِيحُ الِافْتِرَاضِ ثُمَّ فَحْصُ الدَّلِيلِ. -> Dalam final formulation untuk challenging an argument politely, Karena itu, respons terkuat adalah memperjelas asumsi lalu memeriksa bukti.
